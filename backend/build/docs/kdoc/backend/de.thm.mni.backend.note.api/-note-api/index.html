@RestController

class NoteApi(noteRepository: NoteRepository, notebookRepository: NotebookRepository)

REST-Controller für die Verwaltung von Notizen.

Dieser Controller stellt Endpunkte für CRUD-Operationen (Erstellen, Lesen, Aktualisieren, Löschen) für Note-Entitäten bereit. Die Endpunkte sind unter dem Pfad /api/notes verfügbar und erfordern eine Authentifizierung.
