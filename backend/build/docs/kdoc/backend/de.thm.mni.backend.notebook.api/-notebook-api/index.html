@RestController
@RequestMapping(value = ["/api/notebooks"])
@CrossOrigin(origins = ["http://localhost:4200"])
class NotebookApi(notebookRepository: NotebookRepository)

REST-Controller für die Verwaltung von Notizbüchern.

Dieser Controller stellt Endpunkte für CRUD-Operationen (Erstellen, Lesen, Aktualisieren, Löschen) für Notebook-Entitäten bereit. Die Endpunkte sind unter dem Pfad /api/notebooks verfügbar und erfordern eine Authentifizierung.
