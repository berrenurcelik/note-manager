    Class NotesComponent

    Der Component für die Notizen-Seite. Dieser Component ist für das Anzeigen, Filtern und Verwalten von Notizen zuständig. Er lädt Notizen entweder für ein spezifisches Notizbuch oder alle Notizen und ermöglicht die Interaktion (Erstellen, Bearbeiten, Löschen) über einen Dialog.
