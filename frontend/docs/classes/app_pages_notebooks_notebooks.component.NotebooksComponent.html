    Class NotebooksComponent

    Der Haupt-Component für die Notizbuch-Verwaltung. Dieser Component ist verantwortlich für das Laden, Anzeigen und Verwalten der Notizbücher des Benutzers. Er ermöglicht das Hinzufügen, Bearbeiten und Löschen von Notizbüchern über einen Dialog.
